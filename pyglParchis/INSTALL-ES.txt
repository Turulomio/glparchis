Requisitos
==========
Esta aplicación tiene las siguientes dependencias:
- PyQt5
- pyopengl
- python3

Procedimiento para instalar en Linux desde el código fuente
===========================================================
Debes descomprimir el fichero
Una vez estés en el directorio donde Makefile.py está, ejecuta
# python3 Makefile.py

Si usas la distribución Gentoo, te puedes bajar el ebuild desde https://xulpymoney.svn.sourceforge.net/svnroot/xulpymoney/myportage/games-board/glparchis/

Si eres atrevido o quieres ayudarme con el desarrollo te lo puedes bajar del subversion con el siguiente comando
# svn co https://glparchis.svn.sourceforge.net/svnroot/glparchis/pyglParchis glparchis

Para desinstalar utiliza el siguiente comando hace una desinstalación limpia
# python3 Makefile.py --uninstall

Procedimiento para instalar los binarios en Linux
=================================================
Descomprime el fichero con el comando
# tar xvfz glparchis-bin-linux-VERSION.tar.gz

Se entra en el directorio glparchis, que se acaba de crear con
# cd glparchis

y se ejecuta el fichero glparchis con
# ./glparchis
