Requirements
========== 
The app has the following dependencies:
  - PyQt4
  - pyopengl 
  - python2

Procedure to install the game from linux sources
===============================================

Uncompres the file 
Change to the directory where the Makefile is and type
  # make all

If you use Gentoo, you can download the ebuild from 
https://xulpymoney.svn.sourceforge.net/svnroot/xulpymoney/myportage/games-board/glparchis/

If you want to help me, you can get the sourcecode from Sourceforge Subversion typing:
  # svn co https://glparchis.svn.sourceforge.net/svnroot/glparchis/pyglParchis glparchis

 
  # make uninstall 

  
  
Procedure to install the game from linux binaries
================================================

Se descomprime el fichero con el comando 
  # tar xvfz glparchis-bin-linux-VERSION.tar.gz 

Se entra en el directorio glparchis, que se acaba de crear con
  # cd glparchis

y se ejecuta el fichero glparchis con
  # ./glparchis
