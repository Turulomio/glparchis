Requests
========
This application has the following dependencies:
- PyQt5
- pyopengl
- python3

Procedure to install in Linux from sourcecode
=============================================
You must uncompress the file
Once you are in the directory where Makefile.py is, just run
# python3 Makefile.py

Si usas la distribucion Gentoo, te puedes bajar el ebuild desde https://xulpymoney.svn.sourceforge.net/svnroot/xulpymoney/myportage/games-board/glparchis/

Si eres atrevido o quieres ayudarme con el desarrollo te lo puedes bajar del subversion con el siguiente comando
# svn co https://glparchis.svn.sourceforge.net/svnroot/glparchis/pyglParchis glparchis

Para desinstalar utiliza el siguiente comando hace una desinstalacion limpia
# python3 Makefile.py --uninstall

Procedure to install Linux binaries
===================================
Uncompress the file with the command
# tar xvfz glparchis-bin-linux-VERSION.tar.gz

Se entra en el directorio glparchis, que se acaba de crear con
# cd glparchis

y se ejecuta el fichero glparchis con
# ./glparchis

Procedure to install Windows binaries
=====================================
Just run the executable
